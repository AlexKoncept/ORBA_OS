# Code of Conduct / Code de conduite

This project follows the Contributor Covenant v2.1. We expect everyone participating in the Orba OS community to abide by the code of conduct.

Key points:
- Be respectful and civil.
- No harassment, threats, or hateful content.
- When in doubt, treat others with empathy and respect.
- Respect privacy and confidentiality of user data.

Reporting
If you are subject to or witness unacceptable behavior, please report it to: contact@alexkoncept.com

Full text (English): https://www.contributor-covenant.org/version/2/1/code_of_conduct/
